## 1. 系统概述

### 1.1 业务背景

Tailoris 平台面向服装定制产业，版师、设计师在平台上发布原创设计作品（服装纸样、版型设计、图案设计等），需要完善的版权保护机制来保障原创者权益，防止侵权行为。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 版权确权 | 通过区块链存证实现设计作品的版权登记与确权 |
| 侵权检测 | 基于 AI 图像比对技术自动检测平台内相似/侵权作品 |
| 快速维权 | 提供纠纷提交、证据固化、仲裁处理的完整维权流程 |
| 风险防控 | 上传前置检测、水印保护、版权风险责任条款约束 |

### 3.2 文件哈希计算

#### 3.2.1 单文件哈希计算

```java
public class FileHashUtil {

    public static String calculateSHA256(String filePath) throws IOException {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        try (InputStream is = new FileInputStream(filePath);
             DigestInputStream dis = new DigestInputStream(is, digest)) {
            byte[] buffer = new byte[8192];
            while (dis.read(buffer) != -1) {
            }
        }
        return bytesToHex(digest.digest());
    }

    public static String calculateSHA256(byte[] data) throws NoSuchAlgorithmException {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        byte[] hash = digest.digest(data);
        return bytesToHex(hash);
    }

    private static String bytesToHex(byte[] bytes) {
        StringBuilder sb = new StringBuilder();
        for (byte b : bytes) {
            sb.append(String.format("%02x", b));
        }
        return sb.toString();
    }
}
```

#### 3.2.2 多文件合并哈希 (Merkle Tree)

```java
public class MultiFileHashUtil {

    public static String calculateMerkleHash(List<String> filePaths) throws IOException {
        if (filePaths.isEmpty()) {
            throw new IllegalArgumentException("文件列表不能为空");
        }

        List<String> sortedPaths = new ArrayList<>(filePaths);
        Collections.sort(sortedPaths);

        List<String> hashes = new ArrayList<>();
        for (String path : sortedPaths) {
            hashes.add(FileHashUtil.calculateSHA256(path));
        }

        return buildMerkleRoot(hashes);
    }

    private static String buildMerkleRoot(List<String> hashes) {
        if (hashes.isEmpty()) return "";
        if (hashes.size() == 1) return hashes.get(0);

        List<String> nextLevel = new ArrayList<>();
        for (int i = 0; i < hashes.size(); i += 2) {
            String left = hashes.get(i);
            String right = (i + 1 < hashes.size()) ? hashes.get(i + 1) : left;
            String combined = left + right;
            nextLevel.add(FileHashUtil.calculateSHA256(combined.getBytes()));
        }
        return buildMerkleRoot(nextLevel);
    }
}
```

```java
@Service
public class TimestampService {

    public TrustedTimestamp getTrustedTimestamp() {
        Instant now = Instant.now();
        Instant ntpTime = getNTPTime();
        Instant trustedTime = now.isBefore(ntpTime) ? now : ntpTime;

        return TrustedTimestamp.builder()
                .timestamp(trustedTime)
                .timestampNano(System.nanoTime())
                .ntpServer("ntp.aliyun.com")

    }

    private Instant getNTPTime() {
        try {
            NTPUDPClient client = new NTPUDPClient();
            client.setDefaultTimeout(3000);
            InetAddress hostAddr = InetAddress.getByName("ntp.aliyun.com");
            TimeInfo info = client.getTime(hostAddr);
            info.computeDetails();
            return Instant.ofEpochMilli(info.getReturnTime());
        } catch (Exception e) {
            log.warn("NTP时间获取失败，使用系统时间: {}", e.getMessage());
            return Instant.now();
        }
    }
}
```

### 3.4 版权证书生成

#### 3.4.1 证书编号生成规则

```
格式: TLR-CR-{年份}{月份}{日期}-{6位序列号}
示例: TLR-CR-20260527-000001

TLR: Tailoris 平台标识
CR: Copyright Registration 缩写
日期: 登记日期 YYYYMMDD
序列号: 当日递增序列号，6位数字，不足补0
```

```java
@Component
public class CertificateNoGenerator {

    private static final String PREFIX = "TLR-CR-";
    private static final DateTimeFormatter DATE_FORMAT = DateTimeFormatter.ofPattern("yyyyMMdd");

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    public String generate() {
        String dateStr = LocalDate.now().format(DATE_FORMAT);
        String redisKey = "copyright:cert:no:seq:" + dateStr;

        Long seq = redisTemplate.opsForValue().increment(redisKey, 1);
        redisTemplate.expire(redisKey, 2, TimeUnit.DAYS);

        return String.format("%s%s-%06d", PREFIX, dateStr, seq);
    }
}
```

### 4.2 感知哈希算法 (pHash)

#### 4.2.1 pHash 实现

```java
public class PerceptualHashUtil {

    public static long calculatePHash(BufferedImage image) {
        BufferedImage resized = resizeImage(image, 32, 32);
        BufferedImage grayImage = toGrayscale(resized);
        double[][] dctMatrix = computeDCT(grayImage);
        double[][] lowFreq = extractLowFrequency(dctMatrix, 8);
        double median = computeMedian(lowFreq);

        long hash = 0L;
        for (int i = 0; i < 8; i++) {
            for (int j = 0; j < 8; j++) {
                if (lowFreq[i][j] > median) {
                    hash |= (1L << (i * 8 + j));
                }
            }
        }
        return hash;
    }

    public static int hammingDistance(long hash1, long hash2) {
        return Long.bitCount(hash1 ^ hash2);
    }

    public static double similarityPercent(int hammingDistance) {
        return (1.0 - hammingDistance / 64.0) * 100.0;
    }
}
```

### 4.3 CNN 特征提取

```java
@Service
public class CNNFeatureService {

    @Value("${copyright.ai.model.path}")
    private String modelPath;

    private NeuralNetwork featureExtractor;

    @PostConstruct
    public void init() {
        this.featureExtractor = loadPretrainedModel(modelPath);
    }

    public float[] extractFeature(byte[] imageBytes) {
        BufferedImage image = ImageIO.read(new ByteArrayInputStream(imageBytes));
        BufferedImage processed = preprocessImage(image);
        INDArray input = convertToInput(processed);
        INDArray output = featureExtractor.output(input);
        return normalizeFeature(output.toFloatVector());
    }

    private BufferedImage preprocessImage(BufferedImage image) {
        image = resizeImage(image, 224, 224);
        return image;
    }

    public double cosineSimilarity(float[] vec1, float[] vec2) {
        if (vec1.length != vec2.length) {
            throw new IllegalArgumentException("特征向量维度不匹配");
        }
        double dotProduct = 0.0;
        double norm1 = 0.0;
        double norm2 = 0.0;
        for (int i = 0; i < vec1.length; i++) {
            dotProduct += vec1[i] * vec2[i];
            norm1 += vec1[i] * vec1[i];
            norm2 += vec2[i] * vec2[i];
        }
        if (norm1 == 0.0 || norm2 == 0.0) return 0.0;
        return dotProduct / (Math.sqrt(norm1) * Math.sqrt(norm2));
    }
}
```

### 4.4 综合相似度计算

```java
@Service
public class ComprehensiveSimilarityService {

    @Value("${copyright.similarity.weight.phash:0.4}")
    private double pHashWeight;

    @Value("${copyright.similarity.weight.cnn:0.6}")
    private double cnnWeight;

    public SimilarityScore calculateComprehensive(BufferedImage image1, BufferedImage image2) {
        long hash1 = PerceptualHashUtil.calculatePHash(image1);
        long hash2 = PerceptualHashUtil.calculatePHash(image2);
        int hammingDist = PerceptualHashUtil.hammingDistance(hash1, hash2);
        double pHashSimilarity = PerceptualHashUtil.similarityPercent(hammingDist);

        float[] feature1 = cnnFeatureService.extractFeature(imageToBytes(image1));
        float[] feature2 = cnnFeatureService.extractFeature(imageToBytes(image2));
        double cnnSimilarity = cnnFeatureService.cosineSimilarity(feature1, feature2) * 100.0;

        double comprehensive = pHashSimilarity * pHashWeight + cnnSimilarity * cnnWeight;

        return SimilarityScore.builder()
                .pHashSimilarity(round(pHashSimilarity))
                .cnnSimilarity(round(cnnSimilarity))
                .comprehensiveSimilarity(round(comprehensive))
                .hammingDistance(hammingDist)
                .build();
    }
}
```

### 4.5 阈值配置

#### 4.5.1 相似度阈值定义

| 级别 | 阈值范围 | 说明 | 处理方式 |
|------|----------|------|----------|
| 不相似 | 0-40% | 无关联 | 无需处理 |
| 低度相似 | 40-60% | 可能存在共同元素 | 记录备查 |
| 中度相似 | 60-80% | 显著相似 | 标记预警 |
| 高度相似 | 80-90% | 高度相似 | 重点审查 |
| 疑似侵权 | >=90% | 极可能侵权 | 自动判定/人工审核 |
| 前置检测拒绝 | >=95% | 上传时直接拒绝 | 阻止上传 |

```java
@Data
@Configuration
@ConfigurationProperties(prefix = "copyright.similarity")
public class SimilarityThresholdConfig {

    private BigDecimal infringementThreshold = new BigDecimal("90.00");
    private BigDecimal highSimilarityThreshold = new BigDecimal("80.00");
    private BigDecimal mediumSimilarityThreshold = new BigDecimal("60.00");
    private BigDecimal lowSimilarityThreshold = new BigDecimal("40.00");
    private BigDecimal preCheckThreshold = new BigDecimal("95.00");

    public enum SimilarityLevel {
        NONE(0, "不相似"),
        LOW(1, "低度相似"),
        MEDIUM(2, "中度相似"),
        HIGH(3, "高度相似"),
        INFRINGEMENT(4, "疑似侵权");

        private final int code;
        private final String description;
    }

    public SimilarityLevel getLevel(BigDecimal similarity) {
        if (similarity.compareTo(infringementThreshold) >= 0) return SimilarityLevel.INFRINGEMENT;
        if (similarity.compareTo(highSimilarityThreshold) >= 0) return SimilarityLevel.HIGH;
        if (similarity.compareTo(mediumSimilarityThreshold) >= 0) return SimilarityLevel.MEDIUM;
        if (similarity.compareTo(lowSimilarityThreshold) >= 0) return SimilarityLevel.LOW;
        return SimilarityLevel.NONE;
    }
}
```

```java
                new LinkedBlockingQueue<>(1000),
                new ThreadFactoryBuilder().setNameFormat("similarity-check-%d").build(),
                new ThreadPoolExecutor.CallerRunsPolicy()
        );
    }

    public BatchCheckResult batchCheck(String targetImageUrl, List<String> compareImageUrls) {
        BufferedImage targetImage = loadImageFromUrl(targetImageUrl);
        long[] targetHash = new long[]{PerceptualHashUtil.calculatePHash(targetImage)};
        float[] targetFeature = cnnFeatureService.extractFeature(imageToBytes(targetImage));

        List<Future<SingleCheckResult>> futures = new ArrayList<>();
        List<List<String>> batches = Lists.partition(compareImageUrls, batchSize);

        for (List<String> batch : batches) {
            futures.add(executorService.submit(() ->
                processBatch(targetImage, targetHash, targetFeature, batch)));
        }

        List<SingleCheckResult> results = new ArrayList<>();
        for (Future<SingleCheckResult> future : futures) {
            try { results.addAll(future.get().getResults()); }
            catch (Exception e) { log.error("批量检测异常", e); }
        }

        results.sort((a, b) -> b.getSimilarity().compareTo(a.getSimilarity()));

        return BatchCheckResult.builder()
                .targetImageUrl(targetImageUrl)
                .totalChecked(compareImageUrls.size())
                .results(results)
                .highSimilarityCount((int) results.stream()
                    .filter(r -> r.getSimilarity().compareTo(thresholdConfig.getHighSimilarityThreshold()) >= 0)
                    .count())
                .checkTime(new Date())
                .build();
    }
}
```

```java
@Service
public class PreUploadCheckService {

    @Autowired
    private ComprehensiveSimilarityService similarityService;

    @Autowired
    private SimilarityThresholdConfig thresholdConfig;

    @Autowired
    private CopyrightCertificateMapper certificateMapper;

    public PreCheckResult checkBeforeUpload(byte[] imageBytes) {
        String fileHash = FileHashUtil.calculateSHA256(imageBytes);

        CopyrightCertificate exactMatch = certificateMapper.selectOne(
                new LambdaQueryWrapper<CopyrightCertificate>()
                        .eq(CopyrightCertificate::getFileHash, fileHash)
                        .eq(CopyrightCertificate::getDelFlag, 0));
        if (exactMatch != null) {
            return PreCheckResult.builder()
                    .passed(false)
                    .reason("该作品已存在版权登记，请勿重复上传")
                    .matchCertificateNo(exactMatch.getCertificateNo())
                    .similarity(new BigDecimal("100.00"))
                    .build();
        }

        BufferedImage image = bytesToImage(imageBytes);
        long uploadHash = PerceptualHashUtil.calculatePHash(image);
        List<CopyrightCertificate> candidates = findPhashCandidates(uploadHash);

        BigDecimal maxSimilarity = BigDecimal.ZERO;
        CopyrightCertificate maxMatch = null;

        for (CopyrightCertificate candidate : candidates) {
            BufferedImage candidateImage = loadImageFromUrl(candidate.getWorkImages().get(0));
            SimilarityScore score = similarityService.calculateComprehensive(image, candidateImage);
            if (score.getComprehensiveSimilarity().compareTo(maxSimilarity) > 0) {
                maxSimilarity = score.getComprehensiveSimilarity();
                maxMatch = candidate;
            }
        }

        if (maxSimilarity.compareTo(thresholdConfig.getPreCheckThreshold()) >= 0) {
            return PreCheckResult.builder()
                    .passed(false)
                    .reason(String.format("与已有作品【%s】相似度过高(%.2f%%)，疑似侵权",
                            maxMatch.getWorkName(), maxSimilarity))
                    .matchCertificateNo(maxMatch.getCertificateNo())
                    .similarity(maxSimilarity)
                    .build();
        }

        return PreCheckResult.builder().passed(true).maxSimilarity(maxSimilarity).build();
    }

    private List<CopyrightCertificate> findPhashCandidates(long uploadHash) {
        return certificateMapper.selectList(
                new LambdaQueryWrapper<CopyrightCertificate>()
                        .eq(CopyrightCertificate::getCertificateStatus, 1)
                        .isNotNull(CopyrightCertificate::getPhashValue)
        ).stream().filter(cert -> {
            long storedHash = Long.parseLong(cert.getPhashValue());
            return PerceptualHashUtil.hammingDistance(uploadHash, storedHash) <= 10;
        }).collect(Collectors.toList());
    }
}
```

```java
@Service
public class DisputeArbitrationWorkflow {

    @Autowired private CopyrightDisputeMapper disputeMapper;
    @Autowired private ComprehensiveSimilarityService similarityService;

    public enum ArbitrationState {
        SUBMITTED(0, "待处理"),
        EVIDENCE_COLLECTING(1, "证据收集中"),
        AUTO_ARBITRATING(2, "自动仲裁中"),
        MANUAL_REVIEW(3, "人工审核中"),
        ARBITRATED(4, "已裁决"),
        MEDIATED(5, "已调解"),
        WITHDRAWN(6, "已撤诉"),
        REJECTED(7, "已驳回");

        private final int code;
        private final String description;
    }

    @Transactional(rollbackFor = Exception.class)
    public ArbitrationResult executeArbitration(Long disputeId, Long handlerId, String handlerName) {
        CopyrightDispute dispute = disputeMapper.selectById(disputeId);

        SimilarityScore finalScore = calculateFinalSimilarity(dispute);

        if (finalScore.getComprehensiveSimilarity().compareTo(new BigDecimal("95.00")) >= 0) {
            return autoArbitrate(dispute, handlerId, handlerName, finalScore, 1);
        } else if (finalScore.getComprehensiveSimilarity().compareTo(new BigDecimal("80.00")) >= 0) {
            return manualReview(dispute, handlerId, handlerName, finalScore);
        } else {
            return autoArbitrate(dispute, handlerId, handlerName, finalScore, 2);
        }
    }

    @Transactional(rollbackFor = Exception.class)
    public void executeArbitrationResult(Long disputeId, int handleResult, String opinion) {
        CopyrightDispute dispute = disputeMapper.selectById(disputeId);

        disputeMapper.update(null,
                new LambdaUpdateWrapper<CopyrightDispute>()
                        .eq(CopyrightDispute::getId, disputeId)
                        .set(CopyrightDispute::getDisputeStatus, 3)
                        .set(CopyrightDispute::getHandleResult, handleResult)
                        .set(CopyrightDispute::getHandleOpinion, opinion)
                        .set(CopyrightDispute::getHandleTime, new Date()));

        switch (handleResult) {
            case 1: executeInfringementPenalty(dispute); break;
            case 2: clearDispute(dispute); break;
            case 3: executePartialPenalty(dispute); break;
        }
        notifyArbitrationResult(dispute, handleResult, opinion);
    }

    private void executeInfringementPenalty(CopyrightDispute dispute) {
        removeInfringingProduct(dispute);
        recordViolation(dispute);
        freezeAccountIfNecessary(dispute);
    }
}
```

#### 9.2.3 蚂蚁链服务实现

```java
@Service
@Slf4j
public class AntChainService implements BlockchainStoreService {

    @Value("${copyright.blockchain.antchain.endpoint}")
    private String endpoint;

    @Value("${copyright.blockchain.antchain.app-id}")
    private String appId;

    @Value("${copyright.blockchain.antchain.private-key}")
    private String privateKey;

    @Value("${copyright.blockchain.antchain.contract-address}")
    private String contractAddress;

    @Autowired
    private CopyrightBlockchainRecordMapper blockchainRecordMapper;

    @Override
    public BlockchainStoreResult store(Long certificateId, String fileHash, Map<String, Object> txData) {
        try {
            StoreEvidenceRequest request = buildStoreRequest(certificateId, fileHash, txData);
            AntChainClient client = createClient();
            StoreEvidenceResponse response = client.storeEvidence(request);

            return BlockchainStoreResult.builder()
                    .success(true)
                    .txHash(response.getTxHash())
                    .contractAddress(contractAddress)
                    .storeTime(new Date())
                    .message("存证成功")
                    .build();
        } catch (Exception e) {
            log.error("蚂蚁链存证失败: certificateId={}", certificateId, e);
            return BlockchainStoreResult.builder()
                    .success(false)
                    .errorMessage("存证失败: " + e.getMessage())
                    .build();
        }
    }

    private StoreEvidenceRequest buildStoreRequest(Long certificateId, String fileHash, Map<String, Object> txData) {
        CopyrightCertificate cert = certificateMapper.selectById(certificateId);

        Map<String, Object> evidence = new HashMap<>();
        evidence.put("certificateId", certificateId);
        evidence.put("certificateNo", cert.getCertificateNo());
        evidence.put("fileHash", fileHash);
        evidence.put("workName", cert.getWorkName());
        evidence.put("ownerName", cert.getOwnerName());
        evidence.put("issueDate", cert.getIssueDate());
        evidence.putAll(txData);

        StoreEvidenceRequest request = new StoreEvidenceRequest();
        request.setAppId(appId);
        request.setContractAddress(contractAddress);
        request.setEvidenceContent(JSON.toJSONString(evidence));
        return request;
    }

    @Override
    public TransactionStatus queryTransaction(String txHash) {
        AntChainClient client = createClient();
        QueryTxResponse response = client.queryTransaction(txHash);

        return TransactionStatus.builder()
                .txHash(txHash)
                .blockHeight(response.getBlockHeight())
                .blockHash(response.getBlockHash())
                .confirmCount(response.getConfirmCount())
                .txStatus(convertTxStatus(response.getStatus()))
                .txTime(response.getTxTime())
                .build();
    }

    @Override
    public boolean verifyEvidence(String txHash, String fileHash) {
        AntChainClient client = createClient();
        VerifyEvidenceResponse response = client.verifyEvidence(txHash);
        return response.getFileHash().equals(fileHash);
    }

    private AntChainClient createClient() {
        return new AntChainClient(endpoint, appId, privateKey);
    }

    private int convertTxStatus(String status) {
        switch (status) {
            case "PENDING": return 0;
            case "CONFIRMING": return 1;
            case "CONFIRMED": return 2;
            case "FAILED": return 3;
            default: return 0;
        }
    }
}
```

### 9.5 区块链存证定时任务

```java
@Component
public class BlockchainStoreTask {

    @Autowired private BlockchainServiceFactory serviceFactory;
    @Autowired private CopyrightBlockchainRecordMapper blockchainRecordMapper;
    @Autowired private CopyrightCertificateMapper certificateMapper;

    @Scheduled(cron = "${copyright.blockchain.retry.cron:0 */5 * * * ?}")
    public void retryFailedStore() {
        List<CopyrightBlockchainRecord> records = blockchainRecordMapper.selectList(
                new LambdaQueryWrapper<CopyrightBlockchainRecord>()
                        .eq(CopyrightBlockchainRecord::getTxStatus, 3)
                        .lt(CopyrightBlockchainRecord::getRetryCount,
                                CopyrightBlockchainRecord::getMaxRetryCount)
                        .le(CopyrightBlockchainRecord::getNextRetryTime, new Date())
                        .eq(CopyrightBlockchainRecord::getDelFlag, 0));

        for (CopyrightBlockchainRecord record : records) {
            try { retryStore(record); }
            catch (Exception e) {
                log.error("存证重试失败: recordId={}", record.getId(), e);
                updateRetryRecord(record, e.getMessage());
            }
        }
    }

    @Scheduled(cron = "${copyright.blockchain.confirm.cron:0 */2 * * * ?}")
    public void confirmTransactions() {
        List<CopyrightBlockchainRecord> records = blockchainRecordMapper.selectList(
                new LambdaQueryWrapper<CopyrightBlockchainRecord>()
                        .eq(CopyrightBlockchainRecord::getTxStatus, 1)
                        .lt(CopyrightBlockchainRecord::getConfirmCount,
                                CopyrightBlockchainRecord::getRequiredConfirm)
                        .eq(CopyrightBlockchainRecord::getDelFlag, 0));

        BlockchainStoreService service = serviceFactory.getService();

        for (CopyrightBlockchainRecord record : records) {
            try {
                TransactionStatus status = service.queryTransaction(record.getTxHash());

                blockchainRecordMapper.update(null,
                        new LambdaUpdateWrapper<CopyrightBlockchainRecord>()
                                .eq(CopyrightBlockchainRecord::getId, record.getId())
                                .set(CopyrightBlockchainRecord::getTxStatus, status.getTxStatus())
                                .set(CopyrightBlockchainRecord::getBlockHeight, status.getBlockHeight())
                                .set(CopyrightBlockchainRecord::getBlockHash, status.getBlockHash())
                                .set(CopyrightBlockchainRecord::getConfirmCount, status.getConfirmCount())
                                .set(status.getTxStatus() == 2,
                                        CopyrightBlockchainRecord::getConfirmTime, new Date()));

                if (status.getTxStatus() == 2) {
                    certificateMapper.update(null,
                            new LambdaUpdateWrapper<CopyrightCertificate>()
                                    .eq(CopyrightCertificate::getId, record.getCertificateId())
                                    .set(CopyrightCertificate::getBlockchainTxHash, record.getTxHash())
                                    .set(CopyrightCertificate::getBlockchainStatus, 2));
                }
            } catch (Exception e) {
                log.error("查询交易状态失败: recordId={}", record.getId(), e);
            }
        }
    }
}
```

## 11. 安全与风控策略

### 11.1 文件上传安全

| 安全措施 | 说明 |
|----------|------|
| 文件类型校验 | 仅允许 jpg/png/jpeg/webp 等常见图片格式 |
| 文件大小限制 | 单文件不超过 10MB，最多 10 张 |
| 文件内容校验 | 检测是否为真实图片，防止恶意文件上传 |
| 文件哈希去重 | 相同哈希文件不重复存储 |
| 防盗链 | OSS 签名 URL，限制直接访问 |

### 11.2 版权风控规则

| 规则编号 | 规则描述 | 触发动作 |
|----------|----------|----------|
| R1 | 用户上传作品与已有作品相似度 >= 95% | 拒绝上传，提示侵权风险 |
| R2 | 同一用户短时间内大量上传相似作品 | 限制上传频率，触发人工审核 |
| R3 | 用户被投诉 >= 3 次且投诉成立 | 冻结版权登记功能 |
| R4 | 用户上传已被撤销版权的作品 | 记录违规，通知管理员 |
| R5 | 纠纷中用户恶意提交虚假证据 | 记录违规，扣除信用分 |

### 11.3 数据安全措施

```yaml
copyright:
  security:
    # 图片水印配置
    watermark:
      enabled: true
      text: "Tailoris 版权所有"
      opacity: 0.3
      font-size: 24
      position: center
    # 图片防盗链
    anti-leech:
      enabled: true
      referer-white-list:
        - "*.tailoris.com"
        - "*.tailoris.cn"
      allow-empty-referer: false
    # 数据加密
    encryption:
      # 敏感数据加密存储
      sensitive-encryption: true
      # 算法: AES-256-GCM
      algorithm: AES-256-GCM
```

### 11.4 版权风险责任条款

用户上传作品时须同意以下条款（参考项目根目录 [版师开店申请入驻须知及版权风险责任条款.pdf](file:///f:/Tailor/Tailoris/版师开店申请入驻须知及版权风险责任条款.pdf)）：

1. 用户承诺上传作品为原创或拥有合法版权
2. 用户上传作品不得侵犯第三方知识产权
3. 因用户上传侵权作品导致的纠纷，由用户承担全部法律责任
4. 平台有权对涉嫌侵权作品进行下架处理
5. 用户同意平台对侵权作品采取必要的技术措施（水印、模糊处理等）
6. 用户同意平台在纠纷处理过程中提供相关证据

## 12. 性能优化与监控

### 12.1 缓存策略

| 缓存 Key | 数据类型 | TTL | 用途 |
|----------|----------|-----|------|
| `copyright:cert:{id}` | 证书详情 | 30分钟 | 证书详情查询 |
| `copyright:cert:no:{no}` | 证书编号索引 | 30分钟 | 证书编号快速查询 |
| `copyright:user:{userId}` | 用户证书列表 | 15分钟 | 用户证书列表 |
| `copyright:cert:no:seq:{date}` | 证书编号序列号 | 2天 | 证书编号生成 |
| `copyright:phash:index:{hash}` | pHash索引 | 永久 | pHash粗筛 |
| `copyright:similarity:result:{certId}` | 相似度结果 | 1小时 | 相似度检测缓存 |

### 12.2 数据库查询优化

```sql
-- 优化: 上传前置检测时查询 pHash 相近的记录
-- 使用覆盖索引避免回表
ALTER TABLE copyright_certificate
ADD INDEX idx_phash_cover (phash_value, certificate_no, work_name, certificate_status);

-- 优化: 纠纷列表分页查询
-- 使用游标分页代替 offset
SELECT * FROM copyright_dispute
WHERE id < #{lastId} AND dispute_status = 0
ORDER BY id DESC LIMIT 20;

-- 优化: 区块链失败重试查询
SELECT * FROM copyright_blockchain_record
WHERE tx_status = 3 AND retry_count < max_retry_count
AND next_retry_time <= NOW() AND del_flag = 0;
```
