## 1. 概述

### 1.1 模块边界

多商户体系是 Tailoris 平台的核心业务模块，涵盖商家从入驻到运营的全生命周期管理。本模块包含以下核心子模块：

| 子模块 | 核心职责 | 依赖模块 |
|--------|----------|----------|
| 商家入驻 | 商家资质审核、类型判定、入驻流程 | 用户模块 |
| 店铺运营 | 商品管理、订单处理、售后管理 | 订单模块、协同模块 |
| 担保交易 | 资金托管、阶段付款、资金结算 | 支付系统 |
| 质量保证金 | 保证金管理、违规处罚、信用评分 | 财务模块 |

## 2. 商家入驻模块

### 2.1 商家类型定义

平台支持4种商家类型入驻，每种类型具有不同的资质要求和权限范围：

| 类型编码 | 类型名称 | 资质要求 | 权限范围 | 保证金标准 |
|----------|----------|----------|----------|------------|
| 1 | 个人打版师 | 身份证、技能证书、作品集 | 接单、打版、设计 | 基础：￥2,000 |
| 2 | 工作室 | 营业执照、法人身份证、工作室照片 | 接单、生产、定制 | 基础：￥5,000 |
| 3 | 品牌商 | 营业执照、商标注册证、品牌授权 | 品牌发布、分销、招商 | 基础：￥20,000 |
| 4 | 供应链企业 | 营业执照、生产许可证、质检报告 | 批量生产、供应链管理 | 基础：￥50,000 |

#### 2.3.2 审核类型

| 审核类型 | 编码 | 触发条件 | 审核要点 |
|----------|------|----------|----------|
| 入驻审核 | 1 | 商家首次入驻 | 资质真实性、经营范围匹配 |
| 信息变更审核 | 2 | 商家修改关键信息 | 变更前后对比、新资质验证 |
| 续期审核 | 3 | 商家资质到期续期 | 资质有效性、历史违规记录 |
| 升级审核 | 4 | 商家申请类型升级 | 新增资质、经营能力评估 |

### 2.4 审核任务分配策略

```java
/**
 * 审核任务分配策略
 */
public enum AuditAssignStrategy {

    /**
     * 轮询分配：按审核员负载轮询分配
     */
    ROUND_ROBIN("round_robin"),

    /**
     * 按类型分配：不同类型分配给对应专业的审核员
     */
    BY_MERCHANT_TYPE("by_merchant_type"),

    /**
     * 按负载分配：分配给当前待审核任务最少的审核员
     */
    BY_WORKLOAD("by_workload");

    private final String code;
}
```

**分配规则：**

1. 个人打版师/工作室：分配给初级审核员
2. 品牌商：分配给中级审核员
3. 供应链企业：分配给高级审核员
4. 加急审核：优先分配给空闲审核员

### 2.7 入驻流程核心逻辑

```java
/**
 * 商家入驻服务实现
 */
@Service
public class MerchantServiceImpl implements MerchantService {

    @Override
    @Transactional(rollbackFor = Exception.class)
    public MerchantRegisterResult register(MerchantRegisterRequest request) {
        // 1. 校验用户是否已入驻
        if (userExists(request.getUserId())) {
            throw new BusinessException("该用户已入驻");
        }

        // 2. 校验资质完整性
        validateQualifications(request);

        // 3. 生成商家编码
        String merchantCode = generateMerchantCode(request.getMerchantType());

        // 4. 创建商家记录（待审核状态）
        MerchantInfo merchant = buildMerchantInfo(request, merchantCode);
        merchant.setStatus(0);
        merchantMapper.insert(merchant);

        // 5. 创建审核记录
        MerchantAudit audit = buildMerchantAudit(merchant.getId(), request);
        auditMapper.insert(audit);

        // 6. 分配审核任务
        auditAssignService.assign(audit.getId(), merchant.getMerchantType());

        // 7. 发送审核通知
        notificationService.sendAuditNotification(audit);

        return new MerchantRegisterResult(merchant.getId(), merchantCode);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void audit(MerchantAuditRequest request) {
        // 1. 查询审核记录
        MerchantAudit audit = auditMapper.selectById(request.getAuditId());
        if (audit == null) {
            throw new BusinessException("审核记录不存在");
        }
        if (audit.getAuditStatus() != 0 && audit.getAuditStatus() != 1) {
            throw new BusinessException("该审核已处理");
        }

        // 2. 更新审核记录
        audit.setAuditStatus(request.getApproved() ? 2 : 3);
        audit.setAuditorId(request.getAuditorId());
        audit.setAuditTime(new Date());
        audit.setAuditOpinion(request.getOpinion());
        if (!request.getApproved()) {
            audit.setRejectReason(request.getRejectReason());
        }
        auditMapper.updateById(audit);

        // 3. 更新商家状态
        MerchantInfo merchant = merchantMapper.selectById(audit.getMerchantId());
        if (request.getApproved()) {
            merchant.setStatus(1);
            merchantMapper.updateById(merchant);
            // 生成缴纳保证金通知
            depositService.generatePayNotice(merchant.getId());
        } else {
            merchant.setStatus(0);
            merchant.setRejectReason(request.getRejectReason());
            merchantMapper.updateById(merchant);
        }

        // 4. 发送审核结果通知
        notificationService.sendAuditResult(merchant, request.getApproved());
    }
}
```

#### 3.2.4 库存管理策略

```java
/**
 * 库存扣减策略
 */
@Service
public class InventoryService {

    /**
     * 下单扣减库存（预扣减）
     */
    @Transactional(rollbackFor = Exception.class)
    public boolean deductStock(Long productId, Long skuId, int quantity) {
        // 1. 检查库存是否充足
        StoreProductAttrValue sku = productAttrValueMapper.selectById(skuId);
        if (sku.getStock() < quantity) {
            return false;
        }

        // 2. 扣减SKU库存
        int updated = productAttrValueMapper.deductStock(skuId, quantity);
        if (updated == 0) {
            return false;
        }

        // 3. 扣减商品总库存
        productMapper.deductStock(productId, quantity);

        // 4. 记录库存日志
        inventoryLogMapper.insert(new InventoryLog(
            productId, skuId, -quantity, "下单预扣", System.currentTimeMillis()
        ));

        return true;
    }

    /**
     * 取消订单释放库存
     */
    @Transactional(rollbackFor = Exception.class)
    public void releaseStock(Long productId, Long skuId, int quantity) {
        productAttrValueMapper.addStock(skuId, quantity);
        productMapper.addStock(productId, quantity);
        inventoryLogMapper.insert(new InventoryLog(
            productId, skuId, quantity, "订单取消释放", System.currentTimeMillis()
        ));
    }

    /**
     * 确认收货扣减实际库存
     */
    @Transactional(rollbackFor = Exception.class)
    public void confirmDeduct(Long productId, Long skuId, int quantity) {
        // 预扣减已在下单时完成，此处仅记录日志
        inventoryLogMapper.insert(new InventoryLog(
            productId, skuId, 0, "确认收货实际扣减", System.currentTimeMillis()
        ));
    }
}
```

```java
 * 订单超时处理服务

@Service
public class OrderTimeoutService {

    /**
     * 付款超时：30分钟未付款自动取消
     */
    @Scheduled(cron = "0 */5 * * * ?")
    public void handlePaymentTimeout() {
        Date expireTime = DateUtil.offsetMinute(new Date(), -30);
        List<OrderInfo> expiredOrders = orderMapper.selectList(
            new LambdaQueryWrapper<OrderInfo>()
                .eq(OrderInfo::getOrderStatus, 0)
                .lt(OrderInfo::getCreateTime, expireTime)
        );

        for (OrderInfo order : expiredOrders) {
            order.setOrderStatus(6);
            order.setCancelReason("付款超时自动取消");
            order.setCancelTime(new Date());
            orderMapper.updateById(order);

            // 释放库存
            inventoryService.releaseStock(order.getProductId(), order.getSkuId(), order.getItemCount());

            // 发送通知
            notificationService.sendOrderCancelNotification(order);
        }
    }

    /**
     * 收货超时：发货后15天自动确认收货
     */
    @Scheduled(cron = "0 0 2 * * ?")
    public void handleReceiveTimeout() {
        Date expireTime = DateUtil.offsetDay(new Date(), -15);
        List<OrderInfo> timeoutOrders = orderMapper.selectList(
            new LambdaQueryWrapper<OrderInfo>()
                .eq(OrderInfo::getOrderStatus, 3)
                .lt(OrderInfo::getShippingTime, expireTime)
        );

        for (OrderInfo order : timeoutOrders) {
            order.setOrderStatus(4);
            order.setReceiveTime(new Date());
            orderMapper.updateById(order);

            // 触发结算流程
            settlementService.triggerSettlement(order.getId());
        }
    }
```

```java
 * 担保交易服务实现

@Service
public class EscrowServiceImpl implements EscrowService {

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void createEscrowOrder(Long orderId) {
        OrderInfo order = orderMapper.selectById(orderId);

        // 1. 创建托管记录
        EscrowRecord escrow = new EscrowRecord();
        escrow.setOrderNo(order.getOrderNo());
        escrow.setOrderId(orderId);
        escrow.setBuyerId(order.getUserId());
        escrow.setMerchantId(order.getMerchantId());
        escrow.setEscrowAmount(order.getPayAmount());
        escrow.setEscrowStatus(0); // 待付款
        escrow.setCreateTime(new Date());
        escrowMapper.insert(escrow);

        // 2. 如果是阶段付款订单，创建阶段记录
        if (order.getOrderType() == 4) { // 定制订单
            createPaymentStages(order);
        }
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void paymentCallback(PaymentCallbackDTO callback) {
        // 1. 查询托管记录
        EscrowRecord escrow = escrowMapper.selectByOrderNo(callback.getOrderNo());
        if (escrow == null) {
            throw new BusinessException("托管记录不存在");
        }

        // 2. 更新托管状态
        escrow.setEscrowStatus(1); // 已托管
        escrow.setPayTime(new Date());
        escrow.setPayMethod(callback.getPayMethod());
        escrow.setPaymentNo(callback.getPaymentNo());
        escrowMapper.updateById(escrow);

        // 3. 如果是阶段付款，更新对应阶段状态
        if (callback.getStageNo() != null) {
            OrderPaymentStage stage = paymentStageMapper.selectOne(
                new LambdaQueryWrapper<OrderPaymentStage>()
                    .eq(OrderPaymentStage::getOrderId, escrow.getOrderId())
                    .eq(OrderPaymentStage::getStageNo, callback.getStageNo())
            );
            stage.setPaymentStatus(1);
            stage.setPaymentTime(new Date());
            stage.setPaymentNo(callback.getPaymentNo());
            stage.setThirdPartyNo(callback.getThirdPartyNo());
            paymentStageMapper.updateById(stage);
        }

        // 4. 更新订单状态
        OrderInfo order = orderMapper.selectById(escrow.getOrderId());
        order.setPaymentStatus(2);
        order.setPaymentTime(new Date());
        order.setOrderStatus(1);
        orderMapper.updateById(order);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void settleOrder(Long orderId) {
        // 1. 查询托管记录
        EscrowRecord escrow = escrowMapper.selectByOrderId(orderId);
        if (escrow == null || escrow.getEscrowStatus() != 1) {
            throw new BusinessException("托管状态异常");
        }

        // 2. 计算结算金额（扣除平台服务费）
        BigDecimal platformFee = calculatePlatformFee(escrow.getEscrowAmount());
        BigDecimal settleAmount = escrow.getEscrowAmount().subtract(platformFee);

        // 3. 创建结算记录
        SettlementRecord settlement = new SettlementRecord();
        settlement.setOrderNo(escrow.getOrderNo());
        settlement.setOrderId(orderId);
        settlement.setMerchantId(escrow.getMerchantId());
        settlement.setSettleAmount(settleAmount);
        settlement.setPlatformFee(platformFee);
        settlement.setSettleStatus(0); // 待结算
        settlementMapper.insert(settlement);

        // 4. 执行资金划转
        transferService.transfer(escrow.getEscrowAmount(), settlement);

        // 5. 更新托管状态
        escrow.setEscrowStatus(2); // 已结算
        escrowMapper.updateById(escrow);
    }

    /**
     * 计算平台服务费
     */
    private BigDecimal calculatePlatformFee(BigDecimal amount) {
        // 根据商家类型、商品类目获取费率
        BigDecimal rate = feeConfigService.getFeeRate();
        return amount.multiply(rate).setScale(2, RoundingMode.HALF_UP);
    }
}
```

```java
/**
 * 结算服务实现
 */
@Service
public class SettlementServiceImpl implements SettlementService {

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void triggerSettlement(Long orderId) {
        // 1. 查询订单信息
        OrderInfo order = orderMapper.selectById(orderId);
        if (order == null || order.getOrderStatus() != 4) {
            return;
        }

        // 2. 查询结算周期配置
        MerchantInfo merchant = merchantMapper.selectById(order.getMerchantId());
        Integer settleDays = getSettleDays(merchant);

        // 3. 创建定时结算任务
        Date settleTime = DateUtil.offsetDay(order.getReceiveTime(), settleDays);
        SettlementTask task = new SettlementTask();
        task.setOrderId(orderId);
        task.setOrderNo(order.getOrderNo());
        task.setMerchantId(order.getMerchantId());
        task.setSettleTime(settleTime);
        task.setStatus(0); // 待执行
        settlementTaskMapper.insert(task);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void executeSettlement(Long orderId) {
        // 1. 查询订单和托管信息
        OrderInfo order = orderMapper.selectById(orderId);
        EscrowRecord escrow = escrowMapper.selectByOrderId(orderId);

        // 2. 检查是否有未完成的售后
        if (hasPendingAfterSale(orderId)) {
            return; // 等待售后完成
        }

        // 3. 计算结算金额
        BigDecimal totalAmount = order.getPayAmount();
        BigDecimal platformFee = calculatePlatformFee(totalAmount);
        BigDecimal violationPenalty = calculateViolationPenalty(order.getMerchantId());
        BigDecimal settleAmount = totalAmount.subtract(platformFee).subtract(violationPenalty);

        // 4. 执行划转
        merchantAccountService.addBalance(order.getMerchantId(), settleAmount);
        platformAccountService.addIncome(platformFee);
        if (violationPenalty.compareTo(BigDecimal.ZERO) > 0) {
            violationService.deductFromDeposit(order.getMerchantId(), violationPenalty);
        }

        // 5. 记录结算单
        SettlementRecord record = new SettlementRecord();
        record.setOrderNo(order.getOrderNo());
        record.setOrderId(orderId);
        record.setMerchantId(order.getMerchantId());
        record.setSettleAmount(settleAmount);
        record.setPlatformFee(platformFee);
        record.setViolationPenalty(violationPenalty);
        record.setSettleTime(new Date());
        record.setSettleStatus(1); // 已结算
        settlementMapper.insert(record);

        // 6. 更新托管状态
        escrow.setEscrowStatus(2);
        escrowMapper.updateById(escrow);

        // 7. 更新订单状态
        order.setOrderStatus(5);
        order.setCompleteTime(new Date());
        orderMapper.updateById(order);

        // 8. 发送结算通知
        notificationService.sendSettlementNotification(record);
    }
}
```

```java
        MerchantInfo merchant = merchantMapper.selectById(request.getMerchantId());
        BigDecimal standardAmount = getStandardAmount(merchant.getMerchantType());

        // 2. 创建保证金记录
        MerchantQualityDeposit deposit = new MerchantQualityDeposit();
        deposit.setMerchantId(request.getMerchantId());
        deposit.setDepositAmount(standardAmount);
        deposit.setFrozenAmount(BigDecimal.ZERO);
        deposit.setAvailableAmount(standardAmount);
        deposit.setDepositType(1);
        deposit.setPaymentMethod(request.getPaymentMethod());
        deposit.setPaymentNo(request.getPaymentNo());
        deposit.setPaymentTime(new Date());
        deposit.setStatus(1);
        depositMapper.insert(deposit);

        // 3. 更新商家保证金状态
        merchant.setDepositStatus(1);
        merchantMapper.updateById(merchant);

        // 4. 记录支付流水
        paymentLogService.recordDepositPay(request, deposit);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void deductDeposit(Long merchantId, BigDecimal amount, String reason) {
        // 1. 查询保证金记录
        MerchantQualityDeposit deposit = depositMapper.selectOne(
            new LambdaQueryWrapper<MerchantQualityDeposit>()
                .eq(MerchantQualityDeposit::getMerchantId, merchantId)
                .eq(MerchantQualityDeposit::getStatus, 1)
        );
        if (deposit == null) {
            throw new BusinessException("保证金记录不存在");
        }

        // 2. 检查可用金额
        if (deposit.getAvailableAmount().compareTo(amount) < 0) {
            throw new BusinessException("可用保证金不足");
        }

        // 3. 扣除保证金
        deposit.setAvailableAmount(deposit.getAvailableAmount().subtract(amount));
        deposit.setFrozenAmount(deposit.getFrozenAmount().add(amount));

        // 4. 记录扣除记录
        List<DeductRecord> records = JSONUtil.toList(
            deposit.getDeductRecords(), DeductRecord.class);
        if (records == null) {
            records = new ArrayList<>();
        }
        records.add(new DeductRecord(amount, reason, new Date()));
        deposit.setDeductRecords(JSONUtil.toJsonStr(records));

        depositMapper.updateById(deposit);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void refundDeposit(Long merchantId) {
        MerchantInfo merchant = merchantMapper.selectById(merchantId);
        if (merchant.getStatus() != 3) {
            throw new BusinessException("只有已注销商家可退还保证金");
        }

        MerchantQualityDeposit deposit = depositMapper.selectOne(
            new LambdaQueryWrapper<MerchantQualityDeposit>()
                .eq(MerchantQualityDeposit::getMerchantId, merchantId)
        );
        if (deposit == null) {
            throw new BusinessException("保证金记录不存在");
        }

        // 检查是否有未完成的纠纷
        if (hasPendingDispute(merchantId)) {
            throw new BusinessException("存在未完成的纠纷，暂不可退还");
        }

        // 执行退还
        deposit.setStatus(3);
        deposit.setRefundAmount(deposit.getAvailableAmount());
        deposit.setRefundTime(new Date());
        depositMapper.updateById(deposit);

        // 更新商家状态
        merchant.setDepositStatus(2);
        merchantMapper.updateById(merchant);

        // 执行退款
        refundService.executeRefund(deposit);
    }
}
```

```java
    @Override
    @Transactional(rollbackFor = Exception.class)
    public void createViolation(ViolationCreateRequest request) {
        // 1. 创建违规记录
        ViolationRecord record = new ViolationRecord();
        record.setMerchantId(request.getMerchantId());
        record.setViolationCode(request.getViolationCode());
        record.setViolationType(request.getViolationType());
        record.setViolationLevel(request.getViolationLevel());
        record.setViolationDescription(request.getDescription());
        record.setEvidenceFiles(JSONUtil.toJsonStr(request.getEvidenceFiles()));
        record.setPenaltyType(request.getPenaltyType());
        record.setPenaltyAmount(request.getPenaltyAmount());
        record.setPenaltyStatus(1);
        record.setHandlerId(request.getHandlerId());
        record.setHandleTime(new Date());
        violationMapper.insert(record);

        // 2. 发送处罚通知
        notificationService.sendViolationNotification(record);

        // 3. 如果商家未在规定时间内申诉，自动执行处罚
        if (request.getAutoExecute()) {
            scheduleAutoExecute(record.getId());
        }
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void appeal(Long violationId, String appealReason, List<String> appealEvidence) {
        ViolationRecord record = violationMapper.selectById(violationId);
        if (record == null) {
            throw new BusinessException("违规记录不存在");
        }
        if (record.getPenaltyStatus() != 1) {
            throw new BusinessException("当前状态不可申诉");
        }

        record.setAppealStatus(1);
        record.setAppealTime(new Date());
        record.setAppealReason(appealReason);
        violationMapper.updateById(record);

        // 更新处罚状态为申诉中
        record.setPenaltyStatus(2);
        violationMapper.updateById(record);
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void handleAppeal(Long violationId, boolean approved, String handleOpinion) {
        ViolationRecord record = violationMapper.selectById(violationId);
        if (record == null) {
            throw new BusinessException("违规记录不存在");
        }

        if (approved) {
            // 申诉成功，撤销处罚
            record.setAppealStatus(2);
            record.setPenaltyStatus(4);
        } else {
            // 申诉驳回，执行处罚
            record.setAppealStatus(3);
            executePenalty(record);
        }

        record.setPenaltyStatus(3);
        violationMapper.updateById(record);

        // 发送申诉结果通知
        notificationService.sendAppealResultNotification(record, approved);
    }

    /**
     * 执行处罚
     */
    private void executePenalty(ViolationRecord record) {
        // 1. 扣除保证金
        if (record.getPenaltyAmount().compareTo(BigDecimal.ZERO) > 0) {
            depositService.deductDeposit(
                record.getMerchantId(),
                record.getPenaltyAmount(),
                "违规处罚：" + record.getViolationType()
            );
        }

        // 2. 执行其他处罚措施
        switch (record.getPenaltyType()) {
            case 1: // 警告
                // 仅记录，不执行额外操作
                break;
            case 2: // 罚款（已在扣除保证金中处理）
                break;
            case 3: // 下架商品
                productService.offlineAllProducts(record.getMerchantId());
                break;
            case 4: // 降权
                merchantService.reduceWeight(record.getMerchantId());
                break;
            case 5: // 封店
                shopService.freezeShop(record.getMerchantId());
                break;
            case 6: // 清退
                merchantService.cancelMerchant(record.getMerchantId());
                break;
        }
    }
}
```

```java
/**
```

#### 8.3.2 缓存策略

```java
/**
 * 缓存使用策略
 *
 * L1: Caffeine 本地缓存
 *   - 商家信息：5分钟过期
 *   - 店铺信息：5分钟过期
 *   - 权限信息：30分钟过期
 *
 * L2: Redis 分布式缓存
 *   - 商家信息：30分钟过期
 *   - 商品信息：1小时过期
 *   - 订单详情：30分钟过期
 *   - 配置数据：24小时过期
 */
@Service
public class MerchantServiceImpl implements MerchantService {

    @Autowired
    private Cache<String, MerchantInfo> merchantInfoCache;

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    @Override
    public MerchantInfo getMerchantInfo(Long merchantId) {
        String cacheKey = "merchant:info:" + merchantId;

        // L1: 本地缓存
        MerchantInfo merchant = merchantInfoCache.getIfPresent(cacheKey);
        if (merchant != null) {
            return merchant;
        }

        // L2: Redis缓存
        merchant = (MerchantInfo) redisTemplate.opsForValue().get(cacheKey);
        if (merchant != null) {
            merchantInfoCache.put(cacheKey, merchant);
            return merchant;
        }

        // L3: 数据库
        merchant = merchantMapper.selectById(merchantId);
        if (merchant != null) {
            redisTemplate.opsForValue().set(cacheKey, merchant, 30, TimeUnit.MINUTES);
            merchantInfoCache.put(cacheKey, merchant);
        }

        return merchant;
    }

    @Override
    @Transactional(rollbackFor = Exception.class)
    public void updateMerchantInfo(MerchantUpdateRequest request) {
        // 更新数据库
        MerchantInfo merchant = merchantMapper.selectById(request.getMerchantId());
        // ... 更新逻辑
        merchantMapper.updateById(merchant);

        // 删除缓存（Cache Aside模式）
        String cacheKey = "merchant:info:" + merchant.getId();
        merchantInfoCache.invalidate(cacheKey);
        redisTemplate.delete(cacheKey);
    }
}
```

#### 8.3.3 读写分离

```java
/**
 * 读写分离策略
 *
 * 使用 dynamic-datasource 实现读写分离：
 * - 写操作：@DS("master")
 * - 读操作：@DS("slave")
```

#### 8.3.4 分布式锁

```java
/**
 * 分布式锁使用场景
 *
 * 1. 库存扣减：防止超卖
 * 2. 订单支付：防止重复支付
 * 3. 保证金操作：防止并发扣除
 * 4. 结算执行：防止重复结算
 */
@Service
public class InventoryService {

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    /**
     * 库存扣减（分布式锁保护）
     */
    public boolean deductStock(Long productId, Long skuId, int quantity) {
        String lockKey = "lock:stock:deduct:" + skuId;
        String lockValue = UUID.randomUUID().toString();

        try {
            // 获取分布式锁
            Boolean acquired = redisTemplate.opsForValue()
                .setIfAbsent(lockKey, lockValue, 10, TimeUnit.SECONDS);

            if (acquired == null || !acquired) {
                throw new BusinessException("系统繁忙，请稍后重试");
            }

            // 执行库存扣减
            return doDeductStock(productId, skuId, quantity);
        } finally {
            // 释放锁（Lua脚本保证原子性）
            String script = "if redis.call('get', KEYS[1]) == ARGV[1] then " +
                "return redis.call('del', KEYS[1]) else return 0 end";
            redisTemplate.execute(
                new DefaultRedisScript<>(script, Long.class),
                Collections.singletonList(lockKey),
                lockValue
            );
        }
    }
}
```

## 10. 安全与风控

### 10.1 接口安全

| 安全措施 | 实现方式 | 说明 |
|----------|----------|------|
| 身份认证 | JWT Token | 所有接口需要携带有效Token |
| 权限控制 | RBAC | 基于角色的访问控制 |
| 数据签名 | HMAC-SHA256 | 防止请求篡改 |
| 防重放攻击 | Nonce + Timestamp | 请求时间戳校验 |
| 限流 | Redis + 令牌桶 | 防止接口被刷 |

### 10.2 风控规则

| 风控规则 | 触发条件 | 处理措施 |
|----------|----------|----------|
| 频繁退款 | 7天内退款>3次 | 限制下单 + 人工审核 |
| 异常登录 | 异地登录 + 新设备 | 短信验证 |
| 大额交易 | 单笔>￥50,000 | 人工审核 |
| 刷单检测 | 同IP多账号下单 | 标记异常 + 人工审核 |
| 虚假资质 | OCR校验不通过 | 自动拒绝 + 列入黑名单 |

### 10.3 数据加密

| 数据类型 | 加密方式 | 说明 |
|----------|----------|------|
| 密码 | BCrypt | 不可逆加密 |

| 银行卡号 | AES-256 + 脱敏 | 存储加密，展示脱敏 |
